# obsidian

A Claude Code plugin for managing an Obsidian vault.

## Skills

### obsidian-triage

Guides a structured inbox triage session — sorting clippings and fleeting notes into Areas, Reference, Archive, or deletion. Maintains a live Triage Session note in the vault throughout the conversation.

**Triggers automatically when you:**
- Say "let's go through my inbox" or "let's sort these notes"
- Discuss what to do with a clipping or note
- Run `/obsidian-triage`

## Installation

```bash
cc --plugin-dir /path/to/obsidian
```
